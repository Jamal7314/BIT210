SERVICES

Test Centre

We give the users (Managers) the option to register new test centres and test centre officers and also to manage the kit stocks available in each test centre.

Tests

We give the users (Testers) the ability to record new tests and also to record new patients and also to update the result of each test.

Reports

We also provide testing history to each patient for all the tests wich they have taken. Officers also can get the reports of every tests which have been taken in the test centre which they are working.

About Us

There are many Covid-19 test centres that have been set up to manage Covid-19 testing. Covid Testing Information System (CTIS) is required that to allow the health ministry to keep track of tests that have been performed by the various test centres.
